1. Улучшение работы с Зонами S&R

Фильтрация зон по минимальной силе (strength ≥ 65–75)
Учёт возраста зоны (свежие зоны 1–10 дней работают лучше)
Только зоны с HTF confluence (H1 + H4)
Зоны, совпадающие с Weekly/Monthly уровнями
Зоны возле круглых психологических уровней (00, 50)
Фильтр по количеству тестов зоны (чем больше тестов — тем слабее)
Разделение зон на Order Blocks, Breaker Blocks, Mitigation Blocks
Отслеживание "сдвига" зон после сильных новостей

2. Volume Profile (очень рекомендуется)

POC (Point of Control) как приоритетный уровень
Value Area (VAH + VAL)
HVN (High Volume Nodes) — усиление твоих зон
LVN (Low Volume Nodes) — зоны ускорения
Session Volume Profile (London / NY)
Fixed Range vs Visible Range Volume Profile
Сравнение текущего профиля с предыдущими днями
